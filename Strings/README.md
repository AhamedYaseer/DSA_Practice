# Strings

This folder contains string-based DSA problems.

## Common Patterns Used (till date)

* String Traversal
* Simulation
* Efficient String Construction (List + Join)
* Two Pointers
* Vertical Scanning (Prefix Comparison)
* Mathematical Reversal

## Problems Covered

| Problem                                  | Approach / Pattern                                                             |
| ---------------------------------------- | ------------------------------------------------------------------------------ |
| Palindrome Number                        | String Conversion & Reversal, Mathematical Reversal                            |
| Check if a String Is an Acronym of Words | String Traversal, Optimized String Traversal                                   |
| Merge Strings Alternately                | String Concatenation (Simulation), Efficient String Construction (List + Join) |
| Roman to Integer                         | Simulation with Conditional Subtraction                                        |
| Is Subsequence                           | Search & Slicing (Simulation), Two Pointers                                    |
| Longest Common Prefix                    | Vertical Scanning (Prefix Comparison)                                          |
